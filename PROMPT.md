En la administracion publica Española existe el concepto de sede electronica que tiene que facilitar la interacion entre el Contribuyente o Administrado y la Administracion.

El usuario de este sistema debe de poder autenticarse en el sistema. Ahora lo haremos con magic link.

Cuando se entra al sistema podremos presentar la instancia general de momento.
- instancia general (nombre direccion fiscal, nombre del representante sis existe, expone textarea, solicita textarea, add ficheros)

El sistema tiene que tener dos roles: funcionario, administrado.

El administrado podra presentar y ver lo que ha presentado previamente.

El funcionario padra ver lo que han presentado todos los administrados y podra crear un expediente asociado a un registro si cree conveniente.

El expediente tendra un codigo una fecha d creacion el contribuyente sujeto y un tipo de expediente. El funcionario podra hacer actuaciones sobre el expediente.

Las actuaciones tendran una fecha y un texto.

El sistema debe de actuar como saas de tal manera que el funcionario pueda personalizar datos de la home.
